# Tag 标签

# 使用方法

<template>
  <vui-tag type="gorgeous" hue="primary">标签一</vui-tag>
</template>

<script>
import { Tag } from '@game/vui';

export default {
  components: {
    VuiTag: Tag
  },
  ...
};
</script>

# 属性

名称 类型 必填 默认值 描述
type gorgeous | plain | regular N regular 标签类型
hue primary | success | fail | regular N regular 标签色调
size big | regular | small | mini N regular 标签尺寸大小
